# SellClip AI Flutter

Flutter client shell cho SellClip AI.

## Chay app

```powershell
flutter pub get
flutter run --dart-define=API_BASE_URL=http://localhost:8080
```

Android emulator:

```powershell
flutter run --dart-define=API_BASE_URL=http://10.0.2.2:8080
```

Neu can sinh day du thu muc platform:

```powershell
flutter create . --platforms=android,ios,web,windows
```
